Welcome to BarelyACMS! This script will help you configure the first time setup of your BAC managed content. There are only two steps required - that you set up an administrative password, and that you define the page and content container structure of your site. The containers you define here will be accessible by the admin user.




Define pages

Define the page structure of your website below. For each page, you can add any number of containers. These containers are the atomic building blocks of a page. They should represent user-editable content that is present on the page. For example: a description, tagline, phone number or any text content held inside a DIV.
